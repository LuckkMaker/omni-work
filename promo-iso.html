功能特性 · Features

01

Flash 烧录

一键编程、擦除、校验、回读，支持复位运行与空白检查。Hex/Bin 双格式，扇区级擦除控制。

ProgramEraseVerifyRead Back
02

Commander 调试

基于 pyOCD 的交互式命令调试，反汇编、断点、单步执行，寄存器与内存实时读写。

pyOCDDisasmBreakpointStep
03

RTT Viewer

SEGGER RTT 实时双向通信，ASCII / HEX 双视图显示，Omnishell 交互式命令行。

SEGGER RTTASCIIHEXOmnishell
04

Monitor 监视

ELF 符号自动解析，变量实时波形采样，最高 100kHz 采样率，支持 CSV 导出复盘。

ELF 解析100kHz波形CSV
05

Map Analyzer

链接映射文件可视化分析，ROM / RAM 占用统计、模块分类占比，定位内存瓶颈。

.map 解析ROMRAM模块分类
06

Number Converter

32 位多进制转换（DEC / HEX / BIN / OCT），位域可视化，字节拆解，一键复制。

32-bit多进制位域字节拆解
